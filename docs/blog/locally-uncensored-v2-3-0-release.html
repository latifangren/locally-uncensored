v2.3.0 — ComfyUI Plug & Play, Image-to-Video, and Model Bundles

This is the biggest update to Locally Uncensored since launch. ComfyUI integration that actually works without config files. Image-to-Image and Image-to-Video generation. One-click model bundles. And support for models that dropped this week — GLM 5.1, Qwen 3.5, Gemma 4.

Here’s what’s new and why it matters.

ComfyUI Without the Config Hell

If you’ve tried ComfyUI before, you know the drill. Clone the repo, set up a Python environment, download models to the right folders, find workflow JSONs, install custom nodes, hope nothing breaks when you update. It’s powerful software, but the setup cost is real.

v2.3.0 wraps all of that into auto-detection and one-click install. When you open the Create tab, the app checks if ComfyUI is already on your machine. If it is, it finds it automatically. If not, one click handles the installation. No terminal commands, no Python path issues, no manual model file management.

Under the hood, a Dynamic Workflow Builder runs 14 different pipeline strategies. It checks which custom nodes you have installed and builds the correct workflow automatically. You pick a model, write a prompt, and hit Generate. The app figures out the rest.

Image-to-Image

Upload a source image, set a denoise strength between 0.0 and 1.0, write what you want changed. Works with every supported image model — SDXL, FLUX, Z-Image. Low denoise keeps the original composition and adjusts details. High denoise reimagines the image from scratch using your prompt as guidance.

This was the most requested feature after text-to-image landed. Simple drag-and-drop interface, no workflow editing required.

Image-to-Video on 6 GB VRAM

Image-to-Video (I2V) support ships with three backends: FramePack F1, CogVideoX, and SVD.

FramePack is the standout. It uses next-frame prediction to generate video from a single image, and it runs on 6 GB VRAM. That’s a GTX 1660 or an RTX 3060 — hardware that most people already have. Upload an image, get a video. No cloud GPU rental, no queue, no per-second billing.

CogVideoX 5B handles higher-quality output if you have the VRAM for it. SVD-XT is there for the Stable Video Diffusion workflow.

Model Bundles

Instead of hunting for checkpoint files, figuring out which VAE matches which model, and hoping download links aren’t dead — we’re building one-click download bundles. Each bundle includes everything needed: checkpoint, VAE, text encoder, LoRAs where applicable.

Verified bundles are marked. Untested ones show “Coming Soon” until we’ve confirmed they work end-to-end. The list keeps growing — we’re adding more as we verify them.

VRAM-aware filtering sorts bundles into three tabs: Lightweight, Mid-Range, and High-End. Based on your actual GPU VRAM. No more downloading a model to find out your card can’t load it.

Z-Image — Uncensored Image Generation

Z-Image Turbo generates images in 8–15 seconds with zero content filters. No safety classifiers, no prompt rejection. It does what you tell it. Available as both Turbo (speed) and Base (quality) variants.

Works for text-to-image and the new image-to-image pipeline. If unrestricted image generation is what you need, this is it.

New LLM Support

GLM 5.1 shipped two days ago — 754 billion parameters, MoE architecture, MIT license. Built for agentic workflows with sustained tool calling across hundreds of rounds. We added it to the Discover models section on release day.

Qwen 3.5 and Gemma 4 are also in. The hardware-aware onboarding now recommends models based on your GPU VRAM: Gemma 4 E4B for 4 GB cards, Qwen 3.5 9B for 8 GB, larger variants for bigger setups.

Everything Else in the Box

For anyone new to the project: Locally Uncensored is a desktop app that runs AI entirely on your machine. Chat, coding agent, image generation, video generation — one app, no cloud dependency, no Docker requirement.

  • Uncensored AI chat with 20+ provider presets — 12 local backends (Ollama, LM Studio, vLLM, KoboldCpp, and more) plus cloud fallbacks
  • Coding Agent — reads your codebase, writes files, runs shell commands, up to 20 tool iterations
  • 13 MCP agent tools — web search, file I/O, shell execution, code execution, screenshots, system info
  • A/B model compare — same prompt, two models, side by side with parallel streaming
  • Local benchmarks, RAG, voice chat, memory system, thinking mode, 20+ personas

Built with Tauri v2 (Rust backend). Not Electron. Standalone .exe on Windows, build from source on Linux and macOS.

Bug Fixes and Security

  • SSRF protection added to the localhost proxy
  • npm vulnerabilities patched (Vite updated)
  • FramePack workflow: DualCLIPLoader and VAEEncode fixes, preflight custom node check
  • Z-Image gets its own ModelType and strategy (was misclassified as FLUX 2)
  • Think-Mode guard prevents crashes on models that don’t support thinking
  • All 105 download URLs verified (HTTP 200/302)

Download
